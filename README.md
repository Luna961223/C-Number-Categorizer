# C-Number-Categorizer

## 作品說明
這是用C語言編寫的數字歸類工具，可判定使用者輸入的數字最適合用何種資料類別儲存。

## 特色功能
- **支援多種格式：**支援整數 (Integer)、浮點數 (Floating-point)與科學記號（如：1.2e5）的輸入與判定。
- **格式指定碼：**提供資料類別所對應的格式指定碼 (Format Specifier)，方便開發者參考。
- **新手友好：**內建 `help` 指令，提供簡易的操作指引，說明使用方式。
- **重新啟動：**自動詢問是否重啟程式，無需反覆執行程式即可重複使用。

## 開發環境
- **編譯器：**建議使用支援**C99**標準含以上之編譯器。
- **編譯建議：**`gcc main.c -o main -lm`
